Master's Thesis Project: Galaxy Classification with CNN and ViT Models

This project page provides details of my Master’s thesis, including code and documentation for galaxy classification using CNN and ViT models. You can find the full project repository on GitHub here: Master's Thesis Project Repository.

Abstract

The amount of astronomical data collected by observatories has exponentially increased in recent years. Classifying astronomical objects is crucial for processing this data. This thesis contributes to galaxy classification by enhancing accuracy and examining the relationships between data quality, model performance, and observational advancements.

Datasets used include Galaxy10SDSS, Galaxy10DECals, and high-redshift samples of Galaxy10DECals. Preprocessing techniques like Supervised and Unsupervised Wiener Deconvolution, Chan-Vese segmentation, and Non-Local Means denoising were applied. Multi-class classification was performed with Convolutional Neural Networks (CNNs) and Vision Transformers (ViTs). Generally, preprocessing improved model performance. ViT models achieved the best performance in most cases, though CNNs outperformed ViTs on high-redshift samples. Non-Local Means Denoising and Supervised Wiener Deconvolution were top preprocessing methods, with Supervised Wiener Deconvolution proving most efficient in terms of computation.

Methodology

  • Datasets: Galaxy10SDSS, Galaxy10DECals, and high-redshift samples.
  • Preprocessing Techniques: Supervised and Unsupervised Wiener Deconvolution, Chan-Vese segmentation, Non-Local Means Denoising.
  • Models Used: Convolutional Neural Networks (CNNs) and Vision Transformers (ViTs).
  • Key Findings: ViT models generally provided the best performance, except for high-redshift samples where CNNs performed better. Non-Local Means Denoising and Supervised Wiener Deconvolution were the best preprocessing techniques, with Supervised Wiener Deconvolution being the most computationally efficient.
