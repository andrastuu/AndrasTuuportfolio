Key Projects

Galaxy Morphological Classification using Vision Transformers

Developed a highly accurate model for classifying galaxies by morphology using Vision Transformers (ViTs). This project involved intensive data preprocessing, integration of CNNs for initial feature extraction, and the use of deep learning techniques to enhance model accuracy.

The model combines Vision Transformers and Convolutional Neural Networks to process large datasets and improve galaxy classification. This research contributes to advancements in astronomy by automating the process of galaxy identification.

Meme Stock Sentiment Analysis

Analyzed the impact of social media sentiment on the stock market through sentiment analysis of popular "meme stocks." Used NLP and Transformer models to assess the influence of Reddit and Twitter sentiment on market movements.

This project utilized Transformer models and natural language processing tools like VADER and BERT to gather, process, and analyze social media data. The insights gained help to understand how public sentiment can drive financial decisions.

Company URL Scraper

This script scrapes URLs for a list of company names from DuckDuckGo search results. It reads a CSV file with company names, processes them, and outputs the first URL found for each company. The script uses the requests library for HTTP requests and BeautifulSoup for HTML parsing.

Features

  • Cleans and processes company names to remove unwanted symbols.
  • Randomizes User-Agent headers to mimic human-like requests and avoid getting blocked.
  • Adds a delay between requests to avoid being flagged by DuckDuckGo.
  • Outputs the URLs directly to the console.

Repository

GitHub Repository

Prerequisites

  • Python 3.x
  • Required Python libraries:
    • pandas
    • requests
    • beautifulsoup4

Usage

  1. Modify the File Path: Update the file path in the code: df = pd.read_csv('path to your csv file')
  2. Run the Script: python scraper.py
  3. Check the Output: The script will print each company name along with the first URL found from DuckDuckGo.
