Galaxy Morphological Classification using Vision Transformers

Developed a highly accurate model for classifying galaxies by morphology using Vision Transformers (ViTs). This project involved intensive data preprocessing, integration of CNNs for initial feature extraction, and the use of deep learning techniques to enhance model accuracy.

Meme Stock Sentiment Analysis

Analyzed the impact of social media sentiment on the stock market through sentiment analysis of popular "meme stocks." Used NLP and Transformer models to assess the influence of Reddit and Twitter sentiment on market movements.

Company URL Scraper

This script scrapes URLs for a list of company names from DuckDuckGo search results. It reads a CSV file with company names, processes them, and outputs the first URL found for each company. The script uses the requests library for HTTP requests and BeautifulSoup for HTML parsing.

Annual Survey Analysis Dashboard (2022-2023 Comparison)

This dashboard visualizes data from the Annual Survey of 2022 and 2023, focusing on key metrics such as external funding, gender distribution, sectoral representation, and startup growth by cleantech sector. The dashboard helps stakeholders quickly identify trends, gaps, and progress across cleantech sectors, highlighting changes over time.
